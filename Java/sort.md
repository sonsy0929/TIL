## Arrays.sort()

- 정렬방식

  1) Object Array : Tim Sort (Merge Sort + Insertion Sort)
  2) Primitive Array : Dual Pivot Quick Sort

- 시간복잡도

  1. Object Array : O(NlogN)
  2. Primitive Array : O(N^2)
